`app/` contains
1.  `_type/`
2.  `_utility/`
3.  `test/`
-   Purpose : Check if API routes, `app/` and other URI related code work as expected. 
4.  `globals.css`
5.  `layout.tsx`
-   Purpose : Define the common Layout and CSS style of pages.
6.  `page.tsx`
-   Purpose : The main page of our app.
7.  `prev_page.txt`
-   Contains : The `app/page.tsx` code that is generated by `npx create-next-app@latest` commands.
